# toupcam-rs

Hacky library for interfacing with AmScope MU1603/Touptek U3CMOS16000KPA 
USB3.0 CMOS cameras (the VID/PID for this model is `0x0547:0x3016`). 

Don't exactly know what I'm doing with this yet, don't use this.
